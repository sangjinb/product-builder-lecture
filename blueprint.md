# 로또 번호 추첨 사이트

## 개요

이 프로젝트는 사용자가 버튼을 클릭하면 무작위 로또 번호를 생성하고 표시하는 단일 페이지 웹 애플리케이션입니다. 현대적인 웹 기술(HTML, CSS, JavaScript, 웹 컴포넌트)을 사용하여 시각적으로 매력적이고 상호작용이 가능한 사용자 경험을 제공하는 것을 목표로 합니다.

## 기능

*   **번호 생성:** "번호 생성" 버튼을 클릭하면 1부터 45까지의 숫자 중에서 6개의 고유한 번호를 무작위로 선택합니다.
*   **번호 표시:** 생성된 번호는 정렬하여 화면에 표시됩니다.
*   **애니메이션:** 번호가 표시될 때 간단한 애니메이션 효과를 적용하여 시각적 즐거움을 더합니다.
*   **반응형 디자인:** 다양한 화면 크기에서 잘 보이도록 반응형으로 디자인합니다.

## 디자인

*   **색상 팔레트:** 현대적이고 활기찬 느낌을 주는 색상 조합을 사용합니다.
*   **타이포그래피:** 가독성이 좋은 글꼴을 사용하고, 텍스트의 위계를 명확하게 표현합니다.
*   **컴포넌트:** 각 로또 번호는 "공" 모양의 원형 컴포넌트로 시각화하며, 입체감을 주기 위해 그림자 효과를 적용합니다.
*   **레이아웃:** 중앙 정렬된 깔끔한 레이아웃을 사용합니다.

## 현재 구현 계획

1.  **`blueprint.md` 생성:** 프로젝트의 목적, 기능, 디자인 계획을 문서화합니다. (완료)
2.  **`index.html` 업데이트:** 로또 번호 생성기의 기본 구조(제목, 버튼, 번호 표시 영역)를 설정합니다.
3.  **`style.css` 업데이트:** 페이지의 전반적인 레이아웃, 버튼, 번호 표시 영역 등에 현대적이고 미려한 스타일을 적용합니다.
4.  **`main.js` 업데이트:** 번호 생성 로직 및 각 번호를 표시할 `<lotto-ball>` 웹 컴포넌트를 구현합니다.
5.  **검토 및 개선:** 브라우저에서 결과물을 확인하고 기능 및 디자인을 개선합니다.
