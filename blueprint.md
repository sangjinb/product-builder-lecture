# 오늘의 야식 추천 사이트

## 개요

이 프로젝트는 사용자가 버튼을 클릭하면 다양한 야식 메뉴 중 하나를 무작위로 추천해주는 단일 페이지 웹 애플리케이션입니다. 사용자의 '메뉴 결정 장애'를 해결해주고, 즐거운 야식 선택 경험을 제공하는 것을 목표로 합니다.

## 기능

*   **메뉴 추천:** "메뉴 추천받기" 버튼을 클릭하면, 미리 정의된 야식 메뉴 리스트에서 하나의 메뉴를 무작위로 선택하여 보여줍니다.
*   **결과 표시:** 추천된 메뉴는 세련된 카드 형태의 UI를 통해 화면 중앙에 표시됩니다.
*   **애니메이션:** 메뉴가 표시될 때 부드러운 애니메이션 효과를 적용하여 사용자의 기대를 높입니다.
*   **반응형 디자인:** 다양한 화면 크기에서 최적의 화면을 제공합니다.

## 디자인

*   **색상 팔레트:** 식욕을 돋우면서도 야간의 분위기와 어울리는 따뜻하고 매력적인 색상 조합을 사용합니다.
*   **타이포그래피:** 가독성이 좋은 굵은 글꼴을 사용하여 추천 메뉴를 명확하게 전달합니다.
*   **컴포넌트:** 추천 메뉴는 입체적인 그림자 효과가 적용된 카드 UI에 표시하여 사용자의 시선을 집중시킵니다.
*   **레이아웃:** 사용자가 기능에 쉽게 집중할 수 있도록 중앙 정렬된 미니멀한 레이아웃을 사용합니다.

## 현재 구현 계획

1.  **`blueprint.md` 업데이트:** 프로젝트의 목적과 기능을 '야식 메뉴 추천'으로 변경합니다. (완료)
2.  **`index.html` 수정:** 페이지의 제목, 헤더, 버튼 텍스트를 야식 추천 사이트에 맞게 변경합니다.
3.  **`style.css` 수정:** 추천 메뉴를 표시할 카드 UI 등 새로운 디자인 시스템을 적용합니다.
4.  **`main.js` 수정:** 로또 번호 생성 로직을 제거하고, 야식 메뉴를 무작위로 추천하는 로직을 새로 구현합니다.
5.  **GitHub 배포:** 모든 변경사항을 커밋하고 GitHub 원격 저장소에 푸시합니다.
