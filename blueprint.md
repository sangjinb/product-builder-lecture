# 오늘의 야식 추천 사이트

## 개요

이 프로젝트는 사용자가 버튼을 클릭하면 다양한 야식 메뉴 중 하나를 무작위로 추천해주는 단일 페이지 웹 애플리케케이션입니다. 사용자에게 즐거운 야식 선택 경험과 유용한 정보를 제공하여, Google AdSense 승인 기준을 충족하는 '고품질 사이트'로 만드는 것을 목표로 합니다.

## 기능

*   **메뉴 추천:** "메뉴 추천받기" 버튼을 클릭하면, 미리 정의된 야식 메뉴 리스트에서 하나의 메뉴를 무작위로 선택하여 보여줍니다.
*   **결과 표시:** 추천된 메뉴는 이미지, 이름, 그리고 독창적인 설명 문구와 함께 세련된 카드 UI를 통해 화면 중앙에 표시됩니다.
*   **애니메이션:** 메뉴가 표시될 때 부드러운 애니메이션 효과를 적용하여 사용자의 기대를 높입니다.
*   **반응형 디자인:** 다양한 화면 크기에서 최적의 화면을 제공합니다.
*   **다크/라이트 모드:** 사용자의 시각적 편안함을 위해 두 가지 테마를 제공하며, 선택은 브라우저에 저장됩니다.

## 제휴문의 폼 (Partnership Inquiry Form)

*   **기능:** 사용자로부터 이름, 이메일, 문의 내용을 받아 Formspree를 통해 제출합니다.
*   **통합:** Formspree 엔드포인트 `https://formspree.io/f/xeelknrl`를 사용하여 서버 측 코드 없이 제출을 처리합니다.

## 댓글 기능 (Disqus)

*   **기능:** 웹사이트 방문자들이 서로 소통하고 피드백을 남길 수 있는 댓글 섹션을 제공합니다.
*   **통합:** Disqus 플랫폼을 사용하여 댓글 기능을 손쉽게 추가하고 관리합니다.

## Google AdSense 통합

*   **기능:** Google AdSense 광고 스크립트, 계정 메타태그, `ads.txt` 파일을 웹사이트에 삽입하여 수익 창출 기반을 마련하고 AdSense 요구사항을 충족합니다.
*   **`ads.txt`:** `google.com, pub-8670478697060334, DIRECT, f08c47fec0942fa0`
*   **계정 메타태그:** `<meta name="google-adsense-account" content="ca-pub-8670478697060334">`

## AdSense 승인을 위한 사이트 품질 개선 (신뢰성 및 콘텐츠 강화)

*   **콘텐츠 보강:** 각 야식 메뉴에 대한 독창적이고 재미있는 설명 문구를 추가하여 사용자에게 가치있는 정보를 제공합니다.
*   **구조 개선 (푸터):** 사이트 하단에 푸터(Footer)를 추가하여 저작권 정보 및 개인정보처리방침 페이지 링크를 제공합니다.
*   **개인정보처리방침 페이지:** `privacy.html` 페이지를 생성하여 웹사이트의 정보 처리 방침을 명시하고, 이를 통해 사이트의 신뢰도를 높입니다.

## 현재 구현 계획

1.  **`blueprint.md` 업데이트:** AdSense 승인을 위한 사이트 품질 개선 계획을 추가합니다. (완료)
2.  **`main.js` 수정:** `snacks` 데이터 구조에 `description` 필드를 추가하고, 각 메뉴에 대한 설명 문구를 작성합니다. 추천 로직을 수정하여 설명도 함께 표시되도록 합니다.
3.  **`index.html` 수정:** 푸터(Footer) 영역을 추가하고, 개인정보처리방침(`privacy.html`) 페이지로 연결되는 링크와 저작권 정보를 포함합니다. 추천 카드 영역도 설명이 표시될 수 있도록 구조를 수정합니다.
4.  **`style.css` 수정:** 푸터와, 추천 카드에 새롭게 추가된 설명 텍스트 영역의 스타일을 기존 테마와 어울리도록 추가합니다.
5.  **`privacy.html` 파일 생성:** 개인정보처리방침 페이지의 기본 내용을 작성합니다.
6.  **GitHub 배포:** 모든 변경사항을 커밋하고 GitHub 원격 저장소에 푸시합니다.
